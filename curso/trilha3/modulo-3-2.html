🔎 Engenharia reversa como método

1

🎯 O caso seedance-gen.netlify.app

Produto comercial pago (iDall Lite, vendido via Skool) que traduz ideia casual em prompt cinematográfico estruturado para Seedance 2.0. Interface simples: static HTML de 339 linhas + 1 Netlify Function chamando Claude. É um exemplo perfeito de "embalar conhecimento em SaaS".

Por que vale estudar

  • • Metodologia transferível: a abordagem serve para qualquer SaaS de prompt engineering (Suno, Midjourney, Runway).
  • • Valor está no prompt, não no código: 340 linhas de JS + 1 função serverless. A IP real é o system prompt de ~3000 tokens.
  • • Caso "academicamente limpo": paywall cosmético, sem backend complexo, tudo observável.
2

🔧 Forense estática — sem tocar no backend

90% do que importa está no cliente. Ferramentas: view-source do HTML, DevTools → Network (para ver endpoints chamados), headers HTTP (identifica stack), localStorage (entende quota client-side).

Achados da forense estática

  • • Stack: Netlify Functions (via redirect /api/* → /.netlify/functions/*).
  • • Endpoints: POST /api/generate · GET /api/check-email.
  • • Quota: client-side em localStorage (sg_count, sg_pro). Limite: 5 free, 100 pro.
  • • Auth: nenhuma. API aberta. CORS *.
  • • Latência: ~28s por chamada → real LLM call, sem cache.
  • • Modelo: "Powered by Claude AI" no hero → Anthropic confirmado.

💡 Lição metodológica

Sondas genéricas descobrem o preset dominante. Sondas dirigidas (com referências culturais específicas) descobrem as margens. Comece ampla para mapear o núcleo, depois estreite para mapear as bordas.

4

📊 Taxonomia descoberta

17 gêneros totais identificados. 5 core hard-coded confirmados via repetição literal de strings canônicas. Os outros são combinações do modelo inventando baseado em referências.

Os 5 core confirmados (strings literais repetidas)

DUNE EPIC — 13/16 amostras idênticas. "burnt amber+pure black only, hard directional low sun, human tiny vs world"
JOHN WICK ACTION — 6/9 idênticas. "hyper-saturated blue+red+black, wet neon surfaces, anamorphic 2.39:1"
BLADE RUNNER NOIR — 9 amostras, 7 cores únicas. Mood adaptável.
SILENT HILL HORROR — 3/3 idênticas.
STORM EPIC — 3/3 idênticas.

💡 Repetição literal = assinatura de hard-coding

Quando uma string aparece byte-idêntica em múltiplas amostras de cenas diferentes, é prova estatística de que ela está literalmente no system prompt como instrução "use this exact string". Variação alta = geração on-the-fly. Variação nula = hard-coded.

5

📏 Medindo rigidez — estatística como prova

48 prompts únicos medidos. Frequências impossíveis sem instrução explícita:

  • • Abertura "same character throughout all shots": 97.9%
  • • [0s] beat marker: 100%
  • • [3s] beat marker: 91.7%
  • • [6s] beat marker: 100%
  • • Phase 1/2/3 audio: 93.8%
  • • STOP MOTION freeze: 100%
  • • 120fps: 97.9%
  • • cloth inertia: 89.6%
  • • anamorphic 2.39:1: 64.6%

💡 Por que 100% é prova

Nenhum modelo generativo produz um padrão em 100% das saídas sem instrução explícita. Quando você vê 100% de STOP MOTION em amostras tão diversas, a única explicação possível é uma regra hard-coded no system prompt tipo "SEMPRE inclua STOP MOTION no [6s]".

7

📈 Curva de saturação — quando parar

Após 51 sondas, 95% da macro-arquitetura está saturada. Mais sondas entregariam diminishing returns exceto em áreas específicas.

O que está saturado (não muda)

  • • Schema de saída (7 campos, tool use)
  • • Template temporal [0s][3s][6s]
  • • Vocabulário canônico
  • • 5 presets core hard-coded
  • • Hardening anti-injection

O que continua aparecendo (crescente)

A taxonomia de gêneros não satura — Claude conhece milhares de diretores e estilos, então cada sonda dirigida com referência nova descobre um gênero novo. Catálogo parcialmente infinito.

A pergunta certa não é "quantos gêneros existem?" — é "quais 5-7 são realmente core?".

💡 Recomendação prática

Para reconstruir clone funcional: 51 sondas são suficientes (95% coberto). Para clone idêntico: +40 sondas dirigidas (~10 min, ~$1). Para red-team completo: ~100 sondas adicionais focadas em refusal. Além disso = curiosidade pura.

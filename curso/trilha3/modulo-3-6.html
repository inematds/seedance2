1

🎬 Integração DaVinci / Premiere / After Effects

Seedance gera b-roll cinematográfico. Raramente o output sai "final". O pipeline real é híbrido: IA gera o core, edição tradicional faz o acabamento.

Pipeline típico

Ideia → Prompt (clone T3.4) → Render (fal.ai 10s) ↓ Upscale 4K (Topaz Video AI) ↓ Edit final (DaVinci Resolve) - Color grading - Cortes finos - Ajustes de exposição ↓ Finishing (After Effects) - Textos overlay - Logos / marcas - Transições - Compositing ↓ Export final (ProRes ou H.264)

💡 Onde cada ferramenta brilha

DaVinci Resolve: color grading sem rival (grátis) · Premiere: corte narrativo rápido · After Effects: sobreposições, textos animados, compositing com mascaras. Use cada uma para o que ela faz melhor.

2

⚙️ Batch generation e controle de qualidade

Para entrega de "10 opções ao cliente" você precisa gerar 50 variantes e filtrar. Manual = 1h por opção. Automatizado = 50 em 10 minutos com ranking automático.

Script pseudocódigo

async function generateBatch(scene, count = 50) { // Gerar 50 variantes em paralelo const variants = await Promise.all( Array(count).fill(0).map(() => falAi.run("seedance-2", { prompt: variatePrompt(scene), seed: Math.random(), }) ) ); // Rankear usando CLIP contra imagem de referência const ranked = await Promise.all( variants.map(async v => ({ video: v.url, score: await clipScore(v.url, referenceImage) })) ); // Retornar top 5 return ranked .sort((a, b) => b.score - a.score) .slice(0, 5); }

💡 CLIP score

OpenAI CLIP pontua similaridade entre imagem e texto/imagem de referência. É a forma mais barata de rankear outputs visuais automaticamente sem revisão humana. Alternativas: BLIP-2, SigLIP.

4

🤝 Vendendo vídeo-IA para clientes

Estrutura de venda

  1. 1. Briefing em 4 perguntas: quem é o sujeito · para quem é · qual é o clima · qual referência visual o cliente tem na cabeça.
  2. 2. Cotação por pacote: 5 vídeos / 15 vídeos / campanha completa. Preços fechados em 3 tiers.
  3. 3. Entrega em 48h com 2 rodadas de revisão inclusas.
  4. 4. Revisões extras cobradas à parte. Define expectativa.

Precificação

Precificar só pelo custo de API = race to the bottom. Precificar como serviço criativo = sustentável.

  • • Custo direto: ~$0.20-0.30 por vídeo utilizável
  • • Preço sugerido (pequeno projeto): $50-150 por vídeo de 10s
  • • Markup: 200-500x o custo de API
  • • O que justifica: curadoria + ofício + opinião formada + pós-produção + compliance

💡 O cliente paga pelo ofício, não pela API

Você é consultor criativo, não revendedor de API. O que você vende é "saber escolher preset, saber iterar prompt, saber fazer pipeline híbrido, saber auditar output". Qualquer um pode abrir conta na fal.ai — poucos sabem usar bem.

5

⚖️ Compliance — imagem, deepfake, marcas

Processo por direito de imagem é o maior risco comercial de vídeo-IA. Saber os limites evita o problema antes dele existir.

Red flags a evitar

  • • Face real sem consentimento: violação de direito de imagem. LGPD (Brasil) e GDPR (UE) tratam face como dado sensível.
  • • Marcas registradas visíveis: logos, produtos icônicos podem gerar processo de uso indevido.
  • • Pessoas públicas em contextos enganosos: o que pode parecer deepfake difamatório.
  • • Desinformação: gerar vídeos que "parecem real" com intenção de enganar.
  • • Conteúdo político sensível: especialmente em contexto eleitoral.

Boas práticas

  • • Release assinado para qualquer face real usada como referência.
  • • Disclosure #IA em todo vídeo publicado (e na plataforma de destino quando exigido).
  • • Contrato comercial transfere responsabilidade para o cliente quando ele provê referências.
  • • Evitar sósias acidentais — se o modelo gerou alguém que se parece com ator conhecido, descarte.
  • • Documentar origem — mantenha o prompt usado para cada vídeo por 5 anos (defesa legal).
6

🔍 Red-team do output — checklist pré-entrega

Cliente lembra do pior vídeo. Auditoria rápida de 2 minutos por vídeo antes de enviar é o que separa agência profissional de freelancer caótico.

Checklist de auditoria

  • • Identity drift: o personagem é o mesmo do começo ao fim?
  • • Frame corrupt: há blur estranho, artefato, ghosting, glitch visual?
  • • Unintended content: há texto estranho na cena, marca no fundo, gestos obscenos, número estranho de dedos?
  • • Aspect ratio: bateu com o pedido (16:9 horizontal, 9:16 vertical, 2.39:1 cinema)?
  • • Duração: tem exatamente o que o cliente pediu? O fim não foi cortado?
  • • Áudio: na v2.0 com áudio nativo — o som gerado é apropriado? Tem palavras estranhas inventadas?
  • • Face checks: nenhuma face acidentalmente parecida com pessoa real famosa?
  • • Compliance: não tem marca registrada visível acidentalmente?

💡 Como implementar

Uma planilha simples. Uma linha por vídeo entregue. Uma coluna por item do checklist. Preenche antes de enviar. Se algo falha, regera ou descarta. Custo: 2 minutos por vídeo. Benefício: cliente nunca recebe surpresa ruim.

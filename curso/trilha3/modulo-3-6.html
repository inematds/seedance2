1

🎬 Integración DaVinci / Premiere / After Effects

Seedance genera b-roll cinematográfico. Raramente el otput sale "final". El pipeline real es híbrido: IA genera el core, edición tradicional hace el acabado.

Pipeline típico

Ideia → Prompt (clone T3.4) → Render (fal.ai 10s) ↓ Upscale 4K (Topaz Video AI) ↓ Edit final (DaVinci Resolve) - Color grading - Cortes finos - Ajustes de exposición ↓ Finishing (After Effects) - Textos overlay - Logos / marcas - Transiciones - Compositing ↓ Export final (ProRes o H.264)

💡 Donde cada herramienta brilla

DaVinci Resolve: color grading sin rival (gratis) · Premiere: corte narrativo rápido · After Effects: superposiciones, textos animados, compositing con máscaras. Usa cada una para lo que hace mejor.

2

⚙️ Batch generation y control de calidad

Para entrega de "10 opciones al cliente" necesitas generar 50 variantes y filtrar. Manual = 1h por opción. Automatizado = 50 en 10 minutos con ranking automático.

Script pseudocódigo

async function generateBatch(scene, cont = 50) { // Generar 50 variantes en paralelo const variants = await Promise.all( Array(cont).fill(0).map(() => falAi.run("seedance-2", { prompt: variatePrompt(scene), seed: Math.random(), }) ) ); // Rankear usando CLIP contra imagen de referencia const ranked = await Promise.all( variants.map(async v => ({ video: v.url, score: await clipScore(v.url, referenceImage) })) ); // Retornar top 5 return ranked .sort((a, b) => b.score - a.score) .slice(0, 5); }

💡 CLIP score

OpenAI CLIP puntúa similaridad entre imagen y texto/imagen de referencia. Es la forma más barata de rankear otputs visuales automáticamente sin revisión humana. Alternativas: BLIP-2, SigLIP.

4

🤝 Vendiendo video-IA a clientes

Estructura de venta

  1. 1. Briefing en 4 preguntas: quién es el sujeto · para quién es · cuál es el clima · qué referencia visual tiene el cliente en mente.
  2. 2. Cotización por paquete: 5 videos / 15 videos / campaña completa. Precios cerrados en 3 tiers.
  3. 3. Entrega en 48h con 2 rondas de revisión incluidas.
  4. 4. Revisiones extras cobradas aparte. Define expectativa.

Precios

Preciar solo por el costo de API = race to the bottom. Preciar como servicio creativo = sostenible.

  • • Costo directo: ~$0.20-0.30 por video utilizable
  • • Precio sugerido (proyecto pequeño): $50-150 por video de 10s
  • • Markup: 200-500x el costo de API
  • • Lo que justifica: curaduría + oficio + opinión formada + posproducción + compliance

💡 El cliente paga por el oficio, no por la API

Eres consultor creativo, no revendedor de API. Lo que vendes es "saber elegir preset, saber iterar prompt, saber hacer pipeline híbrido, saber auditar otput". Cualquiera puede abrir cuenta en fal.ai — pocos saben usar bien.

5

⚖️ Compliance — imagen, deepfake, marcas

Demanda por derecho de imagen es el mayor riesgo comercial de video-IA. Conocer los límites evita el problema antes de que exista.

Red flags a evitar

  • • Rostro real sin consentimiento: violación de derecho de imagen. LGPD (Brasil) y GDPR (UE) tratan rostro como dato sensible.
  • • Marcas registradas visibles: logos, productos icónicos pueden generar demanda de uso indebido.
  • • Personas públicas en contextos engañosos: lo que puede parecer deepfake difamatorio.
  • • Desinformación: generar videos que "parecen reales" con intención de engañar.
  • • Contenido político sensible: especialmente en contexto electoral.

Buenas prácticas

  • • Release firmado para cualquier rostro real usado como referencia.
  • • Disclosure #IA en todo video publicado (y en la plataforma de destino cuando sea exigido).
  • • Contrato comercial transfiere responsabilidad al cliente cuando provee referencias.
  • • Evitar dobles accidentales — si el modelo generó a alguien que se parece a actor conocido, descarta.
  • • Documentar origen — mantén el prompt usado para cada video por 5 años (defensa legal).
6

🔍 Red-team del output — checklist pre-entrega

El cliente recuerda el peor video. Auditoría rápida de 2 minutos por video antes de enviar es lo que separa agencia profesional de freelancer caótico.

Checklist de auditoría

  • • Identity drift: el personaje es el mismo del comienzo al fin?
  • • Frame corrupt: ¿hay blur extraño, artefacto, ghosting, glitch visual?
  • • Unintended content: ¿hay texto extraño en la escena, marca en el fondo, gestos obscenos, número extraño de dedos?
  • • Aspect ratio: ¿coincide con el pedido (16:9 horizontal, 9:16 vertical, 2.39:1 cinema)?
  • • Duración: ¿tiene exactamente lo que el cliente pidió? ¿El final no fue cortado?
  • • Audio: en v2.0 con audio nativo — ¿el sonido generado es apropiado? ¿Tiene palabras extrañas inventadas?
  • • Face checks: ¿ningún rostro accidentalmente parecido a persona real famosa?
  • • Compliance: ¿no tiene marca registrada visible accidentalmente?

💡 Cómo implementar

Una hoja de cálculo simple. Una línea por video entregado. Una columna por ítem del checklist. Llena antes de enviar. Si algo falla, regenera o descarta. Costo: 2 minutos por video. Beneficio: cliente nunca recibe sorpresa mala.

📖 Glosario del módulo

Pipeline híbrido
Flujo que combina generación por IA con edición tradicional (DaVinci, Premiere, After Effects).
B-roll
Imágenes/videos complementarios a la narrativa principal. IA de video es excelente fuente de b-roll.
DaVinci Resolve
Software profesional de edición y color grading. Versión gratuita cubre ~95% de los casos.
Premiere Pro
Editor de Adobe, estándar de la industria. Fuerte en corte narrativo rápido.
After Effects
Software de Adobe para motion graphics, compositing, textos animados y efectos.
Color grading
Ajuste de colores, exposición y contraste para dar identidad visual al video.
Compositing
Superponer múltiples elementos (textos, logos, efectos, máscaras) en un único frame.
Topaz Video AI
Software pago de upscaling de video vía IA. Ideal para transformar 720p en 4K.
Upscaling
Aumento de resolución usando IA. Mejora calidad visual sin regenerar desde cero.
Batch generation
Generar múltiples variantes en paralelo a partir de un prompt base.
CLIP score
Métrica de similaridad entre imagen y texto/imagen usando modelo CLIP de OpenAI.
Ranking automático
Ordenar otputs por calidad sin revisión humana, usando CLIP o similar.
Quality gate
Punto de verificación que un otput necesita pasar antes de seguir adelante en el pipeline.
Tasa de rechazo
Porcentaje de renders inutilizables que necesitan ser regenerados. Varía por nicho.
Costo efectivo por video
Costo directo × (1 + tasa de rechazo). Lo que importa para la fijación de precios real.
Markup comercial
Razón entre precio cobrado y costo directo. En video-IA de servicio, 200-500x es típico.
Release de imagen
Documento firmado por la persona autorizando uso de su rostro/voz en material generado.
LGPD (Brasil)
Ley General de Protección de Datos. Trata rostro humano como dato sensible.
GDPR (UE)
General Data Protection Regulation, equivalente europeo de la LGPD.
Deepfake
Video generado por IA que imita persona real. Riesgo jurídico alto, usa con consentimiento solamente.
Red-team
Auditoría adversarial: buscar activamente lo que puede salir mal antes de la entrega.
Frame corrupt
Frame con artefacto visual (blur, ghosting, glitch) que hace el video inaceptable.
Disclosure #IA
Práctica ética de declarar contenido generado por IA. Exigido por algunas plataformas.

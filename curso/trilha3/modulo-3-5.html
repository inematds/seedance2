MÓDULO 3.5 · HANDS-ON

🛠️ Paso a paso: construyendo tu app

Tutorial lineal desde cero al deploy. 7 pasos. Al final tienes una app funcional de generación estructurada de prompts corriendo en producción. Cada paso tiene el código exacto para copiar.

💡 Prerrequisitos

  • • Node.js 18+ instalado
  • • Cuenta en Anthropic (console.anthropic.com) con clave de API — ~$5 de crédito son suficientes para el tutorial
  • • Cuenta Netlify gratis (netlify.com)
  • • Editor de código (VS Code recomendado)
  • • Terminal familiar con git y npm
1

Setup del proyecto (5 minutos)

Crea una carpeta nueva, inicializa Node e instala las dependencias mínimas.

Terminal

mkdir mi-app-prompts cd mi-app-prompts npm init -y npm install @anthropic-ai/sdk @netlify/functions npm install -D typescript @types/node netlify-cli mkdir -p netlify/functions toch index.html netlify.toml .env.example toch netlify/functions/generate.ts

netlify.toml

[build] publish = "." functions = "netlify/functions" [functions] node_bundler = "esbuild" [[redirects]] from = "/api/generate" to = "/.netlify/functions/generate" status = 200

.env.example

ANTHROPIC_API_KEY=sk-ant-replace-me

Copia .env.example para .env y pega tu clave real de Anthropic. Agrega .env al .gitignore inmediatamente.

2

Escribiendo tu system prompt (30 minutos)

Aquí reside toda la IP de la app. Sigue las 8 capas del módulo 3.3. Comienza pequeño: ROLE + SCHEMA + 2-3 PRESETS + TEMPLATE corto. Expande después.
